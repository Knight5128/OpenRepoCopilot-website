B · 内置 Agent —— 类 Claude Code 核心循环

当前 app 内嵌的是一套确定性多阶段流水线（analysis-worker.ts）。这里展示的是后续将直接嵌入 Workbench 版本的最终形态：一个完整的、类 Claude Code 的核心 Agent 循环。

1收集上下文
2调用 LLM
3产出工具调用
4执行工具
5结果回灌对话
是否完成？ 否 → 继续循环
是 → 输出结果

工具注册表

  • Read读取文件内容
  • Write写入 / 新建文件
  • Edit精确替换式修改
  • Grep正则全仓检索
  • Glob文件名模式匹配
  • Bash执行命令 / 脚本
  • Task派发子 Agent 分解任务

上下文管理

系统提示设定角色与规范；超长对话自动压缩摘要；复杂任务派发子 Agent 并行探索，结果回主线，避免污染主上下文。
