系统架构

01 / 05

Multi-Agent 分析流水线

基于多 Agent 代码分析管道，自动拆解仓库结构、提取实体关系。分析任务通过队列调度、逐步执行，最终输出完整知识图谱。

多 Agent队列调度自动分析
02 / 05

Knowledge Graph 生成

分析结果写入 knowledge-graph.json，涵盖文件、模块、函数、类、依赖等节点及其关联边，支持社区聚类与层级钻取。

知识图谱社区聚类层级结构
03 / 05

多格式文档导入

支持从 .md、.txt、.pdf、.docx 导入文档知识库，与代码图谱统一建模，实现跨代码与文档的知识关联。

Gunified = Gcode ∪ Gdoc
Ecross = { (ncode, ndoc) | similarity > θ }
文档解析多格式知识融合
04 / 05

Interactive Dashboard

Web 工作台提供图谱搜索、类型过滤、节点详情、Guided Tours 与 Domain Views，支持 PNG / SVG / JSON 导出。

搜索过滤引导导览业务视图导出分享
05 / 05

Local-First 存储架构

所有数据存储在本地 ~/.openrepo-copilot，无需云端依赖。支持 OPENREPO_HOME 自定义路径，图谱产物可提交至 Git 供团队复用。

OPENREPO_HOME=/custom/path
.understand-anything/knowledge-graph.json → git commit
本地存储离线可用团队共享

快速启动

终端

Install
pnpm install
Build
pnpm --filter @understand-anything/core build && pnpm --filter @openrepo-copilot/server build && pnpm --filter @understand-anything/dashboard build
Run
node understand-anything-plugin/packages/openrepo/dist/cli.js ui
